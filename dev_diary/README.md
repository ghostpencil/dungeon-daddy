# Dev Diary

A running log of what it actually feels like to build a real application with an AI coding agent.

Dungeon Daddy is a Python desktop app for tabletop game masters — but it's also a live experiment in AI-assisted software engineering. These entries document what worked, what broke, and what surprised us along the way.

No retrospective polish. Written as it happened.
